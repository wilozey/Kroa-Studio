# KROA Studio MVP Prototype

This workspace now contains a first interactive implementation slice for KROA Studio based on the MVP specification.

Open `index.html` in a browser to try it.

## What is included

- Waitlist capture
- Artist profile snapshot
- Release intake for Singles and EPs
- Metadata validation with warnings and blocking errors
- Artwork and audio checks
- DSP and territory selection
- Royalty split calculator
- Finance snapshot
- Starter AI Release Assistant draft generator

## Current Limits

This is a front-end prototype. It does not yet connect to Supabase, Stripe, OpenAI, S3/R2, email, or a distribution aggregator.
